# Shubham Bhati | Backend Engineer Portfolio

A modern, responsive portfolio website showcasing my work as a Backend Engineer specializing in Java, Spring Boot, Microservices, and AI Integration.

<a href="https://shubh2-0.github.io/" target="_blank">**Visit Portfolio**</a>

## Features

- **Modern Gradient Design** - Blue-purple gradient theme (#667eea to #764ba2)
- **Dark/Light Theme Toggle** - User preference saved in localStorage
- **Fully Responsive** - Optimized for all devices and screen sizes
- **Typing Animation** - Dynamic role display in hero section
- **Project Filtering** - Filter projects by category (Backend, Frontend, Python, AI, Desktop)
- **SEO Optimized** - Meta tags, JSON-LD structured data, sitemap, robots.txt
- **Contact Form** - SMTP.js integration for direct email

## Sections

- **Hero** - Introduction with typing animation and CTAs
- **About** - Tech stack overview and stats
- **Experience** - Timeline design with work history
- **Skills** - Technical skills organized by category
- **Certifications** - 8 key certifications with credential links
- **Featured Projects** - Top 4 projects showcase
- **Projects Page** - All 17 projects with category filters
- **Contact** - Form and contact information

## Tech Stack

**Frontend:**
- HTML5, CSS3, JavaScript
- CSS Custom Properties for theming
- Font Awesome & Devicons
- Google Fonts (Inter, JetBrains Mono)

**Libraries:**
- SMTP.js - Email functionality
- SweetAlert2 - Alert modals

**Hosting:**
- GitHub Pages

## Local Development

1. Clone the repository:
```bash
git clone https://github.com/Shubh2-0/Shubh2-0.github.io.git
```

2. Open `index.html` in a browser or use a local server:
```bash
python -m http.server 8000
```

3. Visit `http://localhost:8000`
